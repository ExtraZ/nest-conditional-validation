# Nest Conditional Validation

A package for conditional validation of DTOs via a custom ValidationPipe and Decorators that assign groups per validation based on predicated supplied via Decorators.

## Install

```
npm i nest-conditional-validation
```

note that `@nestjs/common`, `class-validation` and `class-transformer` are required as peer dependencies.

## Development

```
git clone git@github.com:ExtraZ/nest-conditional-validation.git
cd nest-conditional-validation
npm i
code .
```
